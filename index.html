About Link Intersystems - Commons

Link Intersystems commons (lis) is a collection of reusable software components based on pure Java.

Project Modules

This project has declared the following modules:

Name Description
LIS Commons Math Link Intersystems Commons Math (lis-commons-math) is a reusable software library that provides support for simple common mathematical related tasks.
LIS Commons Lang Link Intersystems Commons Lang (lis-commons-lang) is a library that provides reusable software components based on Java core features under the java.lang package and sub-packages.
LIS Commons Lang Criteria Link Intersystems Commons Lang Criteria (lis-commons-lang-criteria) is a library that provides a criteria api for reflection related tasks.
LIS Commons Beans Link Intersystems Commons Beans (lis-commons-beans) is a collection of reusable software components that provide a simpler programming api for the Java beans api.
LIS Commons Util Link Intersystems Commons Util (lis-commons-util) is collection of reusable software components that extend the standard java.util components.
LIS Commons Test Link Intersystems Commons Test (lis-commons-test) is primarily used by other lis-commons modules to share common test utilities.
LIS Commons Graph Link Intersystems Commons Graph (lis-commons-graph) is a collection of reusable software components that provide support for graph and tree related programming tasks, such as models and search algorithms.
LIS Commons Events Link Intersystems Commons Events (lis-commons-events) is a collection of reusable software components that provide support for event and event listener based programming tasks.
LIS Commons JDBC Link Intersystems Commons JDBC (lis-commons-jdbc) provides an easy api for jdbc meta data access.
LIS Commons JDBC Test Link Intersystems JDBC Test (lis-commons-jdbc-test) provides a JUnit 5 extensions for test database setup and is shipped with the sakila test database provided by mysql.
LIS Commons SQL Link Intersystems Commons SQL (lis-commons-sql) is a small library for generating sql statements and basic dialect support.
LIS Commons SQL Hibernate Link Intersystems Commons SQL Hibernate (lis-commons-sql-hibernate) is an extension for lis-commons-sql that provides SQL statement generation using a hibernate dialect.
LIS Commons Net Pure Java based networking tools.
lis-commons-net-test Link Intersystems commons (lis) is a collection of reusable software components based on pure Java.
LIS Commons Swing Link Intersystems Commons Swing (lis-commons-swing) provides support for Java swing related tasks.
LIS Commons IO Link Intersystems Commons IO (lis-commons-io) provides utilities for java.io.
LIS Commons Security Link Intersystems Commons Security (lis-commons-security) provides security related tools and utilities.
